Deep Dives

Opus 4.7's tokenizer raised your costs without changing the price

Anthropic changed Claude Opus 4.7's tokenizer to improve comprehension. The price per token stayed the same. The catch: identical text now consumes more tokens. OpenRouter's analysis documents the gap. If you migrated production pipelines to Opus 4.7 using cost projections from 4.6, re-run your spreadsheets. The effective price-per-task increased even though the listed price didn't move.

GitHub Actions is a package manager with no lockfile

Every major open-source supply chain incident in the past 18 months traces back to GitHub Actions features behaving exactly as documented. Andrew Nesbitt's analysis: Actions lets workflows pull arbitrary code at build time without version pinning, hash verification, or isolation. Locking deps in package.json means nothing if the CI pipeline building those deps is itself pulling unpinned code. Audit every .github/workflows/ file for pinned SHAs. If you're using @main or @latest in any action reference, you have an unpatched supply chain vulnerability right now.

Agents & Infrastructure

Not conversation history. Structural artifacts. Combined with fleet observability, this is the production agent architecture for 2026: agents that recover from failure and resume across sessions.

Kling MCP — Generate Images & Videos Directly in Claude (@AIwithAmit)

Watch

Kling (AI video generation platform) launched an MCP server that connects to Claude via the Connectors settings in the Claude desktop app. Once connected, you can generate images and videos directly in Claude without switching apps. The setup works well for image generation; video generation was inconsistent on web but worked on desktop. Key tradeoff: MCP connectors burn both Claude tokens AND platform credits simultaneously.

Key takeaways:

  • Kling MCP setup is straightforward: Settings → Connectors → Add Custom Connector → paste MCP server URL → Connect → authorize. Works in Claude desktop app; had issues on web.
  • Double-token cost is the real gotcha: Using an MCP connector consumes Claude tokens and the platform's credits at the same time. For Kling: ~4 credits per image, 72 credits per video, plus Claude token usage on top. Consider whether the convenience is worth the cost vs. using Kling directly.
  • Image generation worked first try, video was inconsistent: Image gen via Kling MCP worked flawlessly on desktop. Video generation tool sometimes failed to load in the same session — required opening a new chat to get video tools to appear.
  • Model selection not exposed: When Claude invoked Kling video generation, it chose Cling 3 automatically. No way to select model from within the Claude prompt.
  • "Always allow" in configure = frictionless but risky: Setting all Kling connector actions to "always allow" removes the approval step. Convenient for demos, but watch credit burn in production use.

5 Underrated Google AI Tools from Google Labs (@PeterYangYT)

Watch

Peter Yang reviews five Google AI Labs products: Pomelo (marketing asset generation), Stitch (UI design), Genie (interactive 3D worlds), Flow (image/video canvas), and NotebookLM. The unifying critique: all tools hit ~90% of the goal but fail on the last 10% because none allow manual edits without re-prompting. Google's core problem is that Labs products are siloed from the main Gemini app.

Key takeaways:

  • Pomelo — brand DNA extraction for marketing: Paste a website URL, Pomelo extracts colors/fonts/images and generates campaigns. Works best for businesses with strong product photography; weak for content/newsletter brands without visual assets.
  • Stitch — design-system-first UI generation: Unlike Claude Artifacts or Vercel v0, Stitch generates a full design system (fonts, colors, components) before producing screens. Uses Nano Banana for placeholder images. Has a predictive click heatmap feature.
  • Stitch gap vs Claude Design: Claude Design asks clarifying questions (focus, vibe, device) before generating. Stitch one-shots from a single prompt. This is a UX regression for non-technical users.
  • Genie — interactive 3D world generation: Real-time frame-by-frame 3D world from text prompt. Impressive tech demo but not a product — missing game loops, progression, and multiplayer.
  • Flow — best Google canvas for video: Powered by Nano Banana + Gemini + Veo together. Key feature: select start and end frames → Flow interpolates the video between them. Fastest speed + best control for generative video.
