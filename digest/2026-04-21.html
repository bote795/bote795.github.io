"Comment and Control" — One Prompt Injection, Three Coding Agents Down

A security researcher demonstrated a single prompt injection pattern that hijacks Claude Code, Gemini CLI, and GitHub Copilot Agent simultaneously. The attack vector: GitHub comments, PR titles, and issue bodies that agents read in the same runtime holding API keys and tokens. Anthropic classified it CVSS 9.4 Critical ($100 bounty). Google paid $1,337. GitHub awarded $500. The architectural flaw is shared across all three: powerful tools and secrets share a runtime with untrusted user input. If you're running coding agents against public repos, audit what they can read and what credentials they hold.
