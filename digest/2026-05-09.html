AI Digest May 9, 2026

Codex turned Chrome into a persistent agent runtime while researchers flagged 5,000+ apps built with AI coding tools shipping exposed API keys. The tooling grows more capable and the security failures more embarrassing at exactly the same rate.

Codex becomes an always-on browser agent

Two Codex updates landed together. Codex runs in Chrome on macOS and Windows, executing across parallel background tabs without taking over your browser. The /goal command persists across terminal restarts, laptop sleep, and 5+ hour pauses. No re-prompting. Close your laptop at midnight, open it at 8 AM, the agent resumes where it stopped. Claude Code has persistent sessions. Codex adds in-browser parallelism on top. First coding agent treating the browser as a durable execution surface.

HTML is replacing markdown for agent output

Thariq Shihipar from the Claude Code team posted "The Unreasonable Effectiveness of HTML" (218 HN points). He stopped generating markdown for "almost everything." Browsers render HTML natively. HTML handles comparisons, diagrams, and color-coding that markdown can't. Claire Vo adopted the pattern the same day. If your agent builds artifacts humans will view, not parse, switch to HTML. Zero rendering step. Higher information density.

Short takes

GPT-Realtime-2 puts reasoning-tier voice in the API. OpenAI shipped three realtime audio models: GPT-Realtime-2 (GPT-5-class reasoning for voice), Realtime-Translate (live multilingual translation), and Realtime-Whisper (streaming transcription). First time OpenAI's reasoning tier runs voice-native. Benchmark this against ElevenLabs and Hume if you build anything with speech.

Claude now lives inside Microsoft 365. GA in Excel, PowerPoint, and Word. Public beta in Outlook. Every enterprise buyer now sees Claude in the toolbar next to Copilot. Distribution play, not a feature.

GitHub admits agent costs are a growing concern. Their Agentic Workflows blog post publishes detailed cost-control patterns. First time GitHub openly names this problem in their own blog. Reference doc for anyone wiring CI agents into repos.

Mozilla used Claude Mythos to find bugs fuzzing missed. Gave Mythos early access to the Firefox codebase. Found and fixed latent security bugs that traditional tooling never caught. Strongest public case for frontier models delivering real security value on production open-source code.

5,000+ vibe-coded apps caught with exposed secrets. No auth, leaked API keys, SQL injection across public apps built with AI tools. deploysafe.io launched to scan for these exact failure modes. Auth, key rotation, parameterized queries before you push. The basics don't change because the code wrote itself.

DeepSeek raising $7.35B per The Information. Largest Chinese AI round ever. Combined with Moonshot's $2B raise (per TechCrunch, 5/7), open-weights-at-one-third-cost now has capital approaching OpenAI's level. Watch for a V5 flagship within the quarter.

  • The agent principal-agent problem. Slop PRs from agent-assisted contributors are an "unmanageable disaster." Fix: give reviewers the agents, not agent output.

The Cowork Setup That Saves Me Hours Every Week (@ThisWeekinAIPodcast)

Watch

Walkthrough of Claude's "Co-work" mode (inside the Claude desktop app) as an alternative to chatbot-style usage. The host argues Co-work is closer to "hiring a fast assistant" than to a search engine — it can run multi-step tasks, touch real local files, and call connected SaaS tools. He covers the full setup (settings, capabilities, connectors, global instructions) and runs two demos: a Gmail-to-Slack triage for guest booking, and a daily funding-news scraper that ships rows into a Notion CRM via a scheduled task.

Key takeaways:

  • Co-work vs. Chat is a different product, not a feature flag: Chat answers one message at a time, no file access, independent replies. Co-work runs multi-step tasks, accesses real files + connected apps, and accepts new messages mid-run. Mindset shift: Treat Claude desktop's Co-work as an employee surface, not a chat surface.
  • The folder on the left is your actual computer — Co-work is not a sandbox or cloud workspace. It reads, writes, and edits real local files. This is the unlock for any "operate on my stuff" workflow.
  • Privacy toggle matters: Turning off "Help improve Claude" prevents your chats from being used to train models. Default is on.
  • Token usage has dual limits: ~5-hour rolling window AND a weekly cap. Cheaper plans hit the weekly cap fast — plan accordingly.
  • Connectors are the leverage point: Connecting Gmail, Calendar, Drive, Notion, Slack etc. is what turns Co-work from "smart chat" into a real assistant. The host considers this the most important setup step.
